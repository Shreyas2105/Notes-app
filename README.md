#Notes App

A simple full-stack Notes Application that allows users to create, edit, view, and delete notes.
The application demonstrates basic CRUD operations using a React frontend and a Node.js + Express backend with MongoDB.

$Features

Create notes with title and content

View all saved notes

Edit existing notes

Delete notes

Data stored in MongoDB database

REST API communication between frontend and backend

#Tech Stack
Frontend

React

Vite

Axios

Backend

Node.js

Express.js

Database

MongoDB (MongoDB Atlas)

#Deployment

Frontend: Netlify

Backend: Render
